# Distributive Candy Bags

A browser-based story deck for explaining the distributive property through candy bags.

## Core idea

Dad brings 3 bags of candy.

Mom brings 5 bags of candy.

Every bag has 6 candies.

The learner first counts separately:

```text
3 * 6 + 5 * 6
```

Then sees that all bags have the same size, so the bag counts can be combined first:

```text
(3 + 5) * 6
```

## Teaching goal

Help the learner understand why a shared multiplier can be factored out:

```text
a * n + b * n = (a + b) * n
```

This is the distributive property, not the associative property.

## Run

Open `index.html` in a browser.

Controls:

- Right arrow / Space / Click: next page
- Left arrow: previous page
- F: fullscreen
