# Multiplication Commutative Array

A browser-based StoryLab lesson for explaining why:

```text
3 * 6 = 6 * 3
```

## Core idea

Start from the meaning of multiplication:

```text
3 * 6 = three groups of six
```

Represent it as a 3-row by 6-column array:

```text
3 rows, 6 blocks in each row
```

Then keep the same 18 blocks fixed and change the grouping:

```text
6 columns, 3 blocks in each column
```

The blocks do not move. Only the way we group and count them changes.

## Teaching goal

Help the learner understand the commutative property of multiplication as a visual re-grouping, not as a rule to memorize first.

## Key transformation

```text
3 rows of 6 -> same rectangle -> 6 columns of 3
```

The animation swaps the two factors while changing row grouping into column grouping.

## Run

Open `index.html` in a browser.

Controls:

- Right arrow / Space / Click: next page
- Left arrow: previous page
- F: fullscreen
- Replay button on the transformation page
- Speed buttons on the transformation page: slow / medium / fast
