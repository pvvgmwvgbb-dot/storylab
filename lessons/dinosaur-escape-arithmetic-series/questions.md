# Teaching Questions

Use these questions to make the learner discover the pattern instead of passively receiving it.

## Before the math

- What do you see?
- Where are we?
- What changed from the museum?
- What danger is coming?
- What do we need to build?

## One-step staircase

- If the ledge is 10 blocks high, is 10 blocks enough?
- Why not?
- How many blocks are in the first row?
- How many blocks are in the second row?
- What is the pattern?
- Can you predict the tenth row?

## Rectangle trick

- What happens if we copy the staircase?
- What happens if we turn the copy around?
- Does the shape become easier to count?
- How many rows does the rectangle have?
- How many blocks are in each row?
- Why do we divide by 2 at the end?

## Two-step staircase

- What changes if each jump rises two blocks?
- What are the row numbers now?
- Is this still a pattern?
- Is it bigger or smaller than the first staircase?
- Why?

## Reflection

- Did we memorize the formula first?
- What did the picture show us?
- Where else might a staircase pattern appear?
