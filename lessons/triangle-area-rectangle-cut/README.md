# Triangle Area: Rectangle Cut

A browser-based story deck for explaining triangle area through a rectangle cut by a diagonal.

## Core idea

Start with a rectangle.

Cut it with one diagonal.

The rectangle becomes two equal right triangles.

So one right triangle is half of the rectangle.

```text
triangle area = base * height / 2
```

Then extend the idea to an ordinary triangle by drawing its height and splitting it into two right triangles.

## Teaching goal

The learner should see that the formula comes from a shape transformation, not from memorization.

## Run

Open `index.html` in a browser.

Controls:

- Right arrow / Space / Click: next page
- Left arrow: previous page
