# Triangle Area: Complete to Rectangle

A browser-based story deck for explaining triangle area by completing a triangle into a rectangle.

## Core idea

Do not start by cutting a rectangle.

Start with a triangle that is hard to calculate.

Complete it with its matching half.

Then the learner can see:

```text
triangle + matching triangle = rectangle
```

So:

```text
triangle area = rectangle area / 2
triangle area = base * height / 2
```

## Extension

For an ordinary triangle, draw the height first.

The height splits the shape into two right triangles.

Each side can be completed into its own rectangle.

The two halves combine back into the same rule:

```text
triangle area = base * height / 2
```

## Teaching goal

The learner should keep seeing the reference shape while the formula appears. The formula should be attached to the picture, not separated from it.
