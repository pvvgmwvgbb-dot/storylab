# Division Monster

A browser-based story deck for explaining division and remainder through a big-mouth monster eating candy.

## Core transformation

```text
total candy -> fixed bite size -> repeated taking away -> number of bites -> remainder
```

## Teaching goal

Help the learner understand what each part of division means:

- Dividend: total candy
- Divisor: candy per bite
- Quotient: number of bites
- Remainder: candy left over, not enough for another full bite

## Run

Open `index.html` in a browser.

Controls:

- Right arrow / Space / Click: next page
- Left arrow: previous page
- F: fullscreen
